API Documentation

Routes

Method Path Description
GET /api/blogs Get all blogs
GET /api/blogs/:id Get a specific blog by ID

PUT /api/blogs/:id Update a specific blog by ID
DELETE /api/blogs/:id Delete a specific blog by ID

Methods

getAuthenticatedUserID(c *gin.Context)

Returns the authenticated user ID extracted from the token.

Parameters:

  • c - The Gin context

Returns:

  • string - The authenticated user ID
  • error - An error if authentication fails
